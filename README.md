# Minnowbrook Analytic Reasoning Seminar 2026

Website for the **Minnowbrook Analytic Reasoning Seminar**, June 3–6, 2026 at Minnowbrook Conference Center, Blue Mountain Lake, NY.

Organized by Kristopher Micinski, Syracuse University.

## Development

This is a static HTML/CSS site — no build tools required. Edit `index.html` and `css/style.css` directly.

To preview locally:

```bash
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).

## Deployment

The site is hosted on **GitHub Pages** and deploys automatically when changes are pushed to the `main` branch via GitHub Actions.

**First-time setup:** Go to repo Settings → Pages → Source → select **"GitHub Actions"**.
